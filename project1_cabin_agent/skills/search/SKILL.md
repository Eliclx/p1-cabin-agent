# Search Domain Knowledge

> 读者：云端 LLM

## 域边界

负责周边 POI 搜索。

| 用户输入 | 归你 | 原因 |
|---------|------|------|
| "附近有没有加油站" | ✓ | 周边搜索 |
| "帮我找附近的医院" | ✓ | 周边搜索 |
| "搜一下火锅店" | ✓ | 周边搜索 |
| "导航到天府广场" | ✗ navigation | 导航 |
| "播放周杰伦" | ✗ media | 媒体搜索 |

## 基础信息

搜索半径默认 5 公里，可调 1-50 公里。当前使用 mock 数据。

## 工具

search_poi: 纯 mock，无外部 API。
